接口可以嵌套在类或其他接口中。

示例

```java
/**
 * @FilePath interfaces.demo.ch9_8/NestingInterfaces.java
 * @Author
 * @Date 1/7/2020 10:56 AM
 * @Version 1.0.0
 * @Description
 */

package interfaces.demo.ch9_8;

class A {
    interface B {
        void f();
    }
    public class BImp implements B {
        public void f() {}
    }
    private class BImp2 implements B {
        public void f() {}
    }
    public interface C {
        void f();
    }
    class CImp implements C {
        public void f() {}
    }
    private class CImp2 implements C {
        public void f() {}
    }
    private interface D {
        void f();
    }
    private class DImp implements D {
        public void f() {}
    }
    public class DImp2 implements D {
        public void f() {}
    }
    public D getD() { return new DImp2(); }
    private D dRef;
    public void receiveD(D d) {
        dRef = d;
        dRef.f();
    }
}

interface E {
    interface G {
        void f();
    }
    // Redundant "public":
    public interface H {
        void f();
    }
    void g();
    // Cannot be private within an interface:
    //! private interface I {}
    // Cannot be protected within an interface:
    //! protected interface J {}
}

public class NestingInterfaces {
    public class BImp implements A.B {
        public void f() {}
    }
    class CImp implements A.C {
        public void f() {}
    }
    // Cannot implement a private interface except
    // within that interface's defining class:
    //! class DImp implements A.D {
    //!  public void f() {}
    //! }
    class EImp implements E {
        public void g() {}
    }
    class EGImp implements E.G {
        public void f() {}
    }
    class EImp2 implements E {
        public void g() {}
        class EG implements E.G {
            public void f() {}
        }
    }
    public static void main(String[] args) {
        A a = new A();
        // Can't access A.D:
        //! A.D ad = a.getD();
        // Doesn't return anything but A.D:
        //! A.DImp2 di2 = a.getD();
        // Cannot access a member of the interface:
        //! a.getD().f();
        // Only another A can do anything with getD():
        A a2 = new A();
        a2.receiveD(a.getD());
    }
} ///:~

 *///:~
```

注意：

1、在类中嵌套接口时，嵌套的接口可拥有public和“包访问”，但也可以限定为private。比如，类A中的接口D。

2、但在接口嵌套接口时，嵌套的接口不可限定为private，因为所有的接口元素都必须是public的，默认就是public。如，接口E中的接口G、H、I。

3、当实现某个接口时，并不需要实现嵌套在其内部的任何接口。如，类EImp、EGImp。

4、类中嵌套的private接口不能在定义它的类之外被实现。比如上面的类A中接口D。
