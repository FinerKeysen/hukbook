## 插件

- Alibaba Java Coding Guidelines，阿里的java开发规范，会实时给出代码意见
- Git commit template，提交模板
- GitToolBox，辅助git
- IDE Eval Reset，evaluate无限续杯，需要添加第三方插件库`https://plugins.zhile.io`
- Rainbow Brackets，括号美化
- Translation，翻译辅助
